Test Clutch Graphs

These are graphs relating to tests run on the main branch of the project (i.e. not pull requests). Many graphs are most useful after clicking on one of the breakdown lines shown below each graph which restricts the graph to show only matching points.

Test Durations

Test job duration
Length of time taken for each run of each test job, which may do more than just testing (wall clock duration).

Time taken to run tests, by test format
Length of time taken to just run the tests of each test job (wall clock duration).

Test Counts

Number of tests run, by test format
How many tests are being run by each test job, split out by test format.

Tests run, by test format and test result
How many tests are being run by each test job, split out by test format and by the test pass/fail status.

Test pass/fail results
How many tests are passing, failing or have other statuses.

Tests that have failed
How many tests that have unexpectedly failed, by test job.

Total tests that have failed
Total number of tests that have unexpectedly failed.

Tests actually run, by format
How many tests that are attempted (not skipped), by test job.

Maximum tests actually run, by format
The maximum number of tests that are attempted (not skipped) in a single job, by test format.

Prometheus Graphs

These are graphs relating to Prometheus itself.

Query rate
Rate of HTTP queries to Prometheus, in queries per second.

Memory usage
Memory size of the Prometheus instance, in bytes.

TSDB size
Amount of space used by the Time Series DataBase, in bytes.
